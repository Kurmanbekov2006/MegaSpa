📦 Mega SPA (Vanilla JS Project)
🚀 Описание

Это мини одностраничное приложение (SPA), написанное на чистых HTML, CSS и JavaScript без фреймворков.

Проект имитирует базовую архитектуру современных фронтенд-приложений:

навигация между страницами
управление состоянием (state)
список задач (Todo)
статистика
сохранение данных в localStorage
🧠 Что реализовано
SPA роутинг без перезагрузки страницы
глобальное состояние приложения
добавление / удаление задач
хранение данных в браузере
компонентный подход (функции UI)
простая архитектура как в современных фреймворках
